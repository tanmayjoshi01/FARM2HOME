## FARM2HOME

Minimal full-stack starter (backend + frontend + database schema).

### Setup and run

1. Open a terminal in the project root:

```bash
cd /d D:\FARM2HOME\FARM2HOME
```

2. Install and run the backend:

```bash
cd backend
npm install
npm start
```

Backend runs at `http://localhost:5000/health`.

3. Install and run the frontend (in a new terminal):

```bash
cd /d D:\FARM2HOME\FARM2HOME\frontend
npm install
npm start
```
